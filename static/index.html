How It Works

A four-step ML pipeline turns raw URLs into safety verdicts.

01

URL Input

You paste a URL. The system normalises it and extracts the raw text.

02

TF-IDF Vectorisation

URL tokens are transformed into a 30,000-dimensional TF-IDF feature vector.

03

Logistic Regression

The trained sigmoid model outputs a probability for scam vs legitimate.

04

Verdict

A clear SAFE / SCAM verdict is returned with a confidence score and URL breakdown.
